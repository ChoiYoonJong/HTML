폼관련태그

form 태그는 html 사용자가 입력할수 있는 양식을 제공하는 태그이다.
form태그내의 input태그들을 통해 사용자가 입력한 정보를 서버로 넘기면서 요청하는 역할을 한다.
* action 속성 : 폼내에 입력된 값들을 전달할 서버측 URL을 기입한다.
* method 속성 : get/post 방식으로 전송 방식을 지정한다.
> get : 서버에 사용자가 입력한 값을 전달하면서 요청시 사용자가 입력한 값들이 url에 노출되는 방식 (전달값에 제한이 있음)
> post : 서버에 사용자가 입력한 값을 전달하면서 요청은 하되 url 에 노출되지않음 ( 전달값을 볼수 없고 , 크기제한 없음)

 

submit 버튼을 눌렀을때 method 요청 전송방식으로 action 에 지정한 경로로 사용자가 입력한값을 (key=value) 전달한다.
